🛒 E-Commerce Website (JSP / Servlet)

A simple e-commerce web application built with JSP, Servlet, and MySQL, deployed on Apache Tomcat.
-------------------------------------
🚀 Features
Storefront

Product listing & categorization

Product detail page

Shopping cart (add / update / remove)

Checkout flow

Order creation & order history

User System

User registration & login

Session-based authentication

Profile management

Admin Panel

Product CRUD

Order management

User management

🏗️ Tech Stack

Backend: Java (JSP / Servlet)

Frontend: HTML, CSS, JavaScript

Database: MySQL

Server: Apache Tomcat
